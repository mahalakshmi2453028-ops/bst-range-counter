# BST Range Counter

This repository contains a Python program to count the number of nodes in a Binary Search Tree (BST) that lie within a given range.

## Description
The program uses recursion and BST properties to efficiently count nodes between a lower and upper bound.

## Language Used
- Python

## How to Run
1. Open the file `count_bst_range.py`
2. Run the program using:
   python count_bst_range.py

## Purpose
This project is created for practicing data structures and algorithms using Python.
